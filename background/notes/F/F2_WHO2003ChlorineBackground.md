## 1. 文档目的

`[原文]` Preface：WHO 为 GDWQ 中各化学物质准备 **health criteria documents** — 评估饮用水中暴露的健康风险，供专家会议确定 **guideline value**。

---

## 2. 中文摘要（结构）

1. **General description**：Cl₂ / HOCl / NaOCl；水中形成 HOCl 与 OCl⁻；用途消毒。  
2. **Analytical methods**：比色法测 free chlorine **0.1–10 mg/L**；MDL **~0.02 mg/L** `[原文]` §Analytical methods — 与 D2 Hach ±0.02 一致。  
3. **Environmental levels**：消毒饮用水中氯 **0.2–1 mg/L** 常见 `[原文]` §Water。  
4. **毒理学**：动物实验；IARC 次氯酸盐 **Group 3**（对人致癌性不可分类）。  
5. **Guideline value**：基于 rodents 2 年 drinking-water 研究 **NOAEL 15 mg/kg/d** → TDI → **5 mg/L** free chlorine（100% 分配）— **健康上限**，不是管网最低 residual `[原文]` §Guideline value。

---

## 3. Guideline value 推导（原文）

| 步骤 | 内容 |
| --- | --- |
| NOAEL | 15 mg/kg body weight per day（2 y 鼠，hypochlorite） |
| UF | 100（种间+种内） |
| TDI | 150 µg/kg |
| 饮水分配 | 100% → **5 mg/L**（rounded） |
| 备注 | 保守；**5 mg/L 以下多数可尝出氯味**（低至 0.3 mg/L） |

`[原文]` — **这与 operational「维持 ≥0.2 mg/L 保微生物屏障」是不同逻辑**（健康上限 vs 消毒保障）。

## 4. 与 0.2 mg/L 项目阈值的关系

| 来源 | 内容 | 本 PDF |
| --- | --- | --- |
| **0.2–1 mg/L** | 消毒水 **常见浓度范围** | ✅ `[原文]` p.2 Water |
| **≥ 0.2 mg/L 管网最低** | WHO GDWQ 消毒实践 / 接触 30 min 后 0.5 mg/L 等 | ❌ **不在本文件** |
| **5 mg/L** | WHO **健康 guideline value** | ✅ |
| **巴西 0.2 min** | D2 Portaria 2914 | 另一司法辖区 |
